* 새로 생겨날 직업들?
  ==> 일자리는 인간의 욕망이다.
  
  - 기계혁명  : 생존의 물질 욕망(1차)
  - 전기혁명  : 안정의 물질 욕망(2차)
  - 인터넷혁명: 사회적 연결 욕망(3차)
  
  ==> 새로운 인간의 욕망은?
  
  ** 메슬로우의 인간의 욕구 5단계설 (욕망과 노동 총량 => 불변의 오류)
     1. 생리적 욕구    -> 기능  (물질 / 기후 적합 / 생리위생 / 1,2차 산업 혁명)
     2. 안전의 욕구    -> 안전  (물질 / 신체 보호            / 1,2차 산업 혁명)
     3. 사회 귀속 욕구          (우리 / 커뮤니케이션 / 사회참가 / 기호화 / 신체장식 / 3차 산업혁명)
     4. 명예 욕구               (나   / 자기 표현 / 4차 산업혁명) 
     5. 자아실현 욕구           (나   / 자기 실현 / 4차 산업혁명)  ==> 이후 고차원의 욕구
